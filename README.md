# Для запуска

### В docker
1. Склонируйте репозиторий и перейдите в директорию с проектом
2. Выполните

```bash
docker-compose up -d
```


### На хосте
1. Склонируйте репозиторий и перейдите в директорию с проектом
2. Установите необходимые зависимости

```bash
pip install -r requirements.txt
```

3. Выполните

```bash
streamlit rum src/app.py
```

4. Перейдите по адресу http://localhost/
